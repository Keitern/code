Animation Workshop

Animating impact frames

For this project the main purpose was to learn animation principles and the use thereof, furthermore it served as an opportunity to learn how to use Adobe Illustrator, as well as html/css to turn some simple spites (illustrations), into fluid animations, aswell as the configuration of said animation, using html and css.

Here I drew about 16 spites in illustrator using the artboard tool, after which I coded it into my html and called to it using css, wherein I also learned how to adjust the playrate aswell as whether or not it loops, and window size.
